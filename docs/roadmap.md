# Roadmap

- V0 - Workspace, fenêtre, event loop, about pages
- V1 - URL, HTTP simple, DocumentLoader
- V2 - HTML tokenizer/tree builder, DOM arena
- V3 - CSS parser, cascade minimale, user-agent stylesheet
- V4 - Layout block/inline minimal
- V5 - Paint display list + wgpu compositor
- V6 - JS lexer/parser/interpreter
- V7 - JS bytecode VM + DOM bindings
- V8 - Inputs, events, forms
- V9 - Fetch, cookies, cache, storage
- V10 - Flexbox, Grid, images, fonts
- V11 - Multi-process architecture
- V12 - DevTools avancés
- V13 - WPT/Test262 integration
- V14 - Performance, incremental layout/paint, GC avancé, JIT

Les TODO de développement restent documentaires et ne doivent pas casser les chemins exécutés par défaut.
