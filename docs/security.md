# Sécurité

La sécurité repose sur le modèle Origin/Site, Same-Origin Policy, CORS, CSP, sandbox d'iframes, politique TLS/certificats, blocage mixed content, permissions et isolation de processus cible.

V0 définit les types centraux. Les versions suivantes connecteront ces politiques au loader réseau, au DOM, aux Web APIs et au process manager.
